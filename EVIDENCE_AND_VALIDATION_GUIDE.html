Show proof a reviewer can rerun

The assignment does not require production infrastructure. It requires evidence that the local system works, fails visibly, blocks unsafe work, and recovers through documented steps.

Minimum evidence set

AreaEvidence
Fresh checkoutExact setup, test, and smoke commands.
Healthy status Output showing all three feeds fresh and eligible.
Stale feed Output showing one stale feed fails closed while the other feeds remain eligible.
RecoveryCommand and output proving eligibility is restored.
GitHub Actions Workflow running setup, tests, smoke, and docs validation.
Observability Freshness signal, liveness signal, page threshold, ticket threshold.

Example local transcript

make setup
make test
make smoke

Strong evidence

  • Identifies the feed, freshness age, blocked action, and recovery signal.
  • Is generated by commands a reviewer can run.
  • Explains page versus ticket thresholds.
  • States what is simulated, what is real, and what was intentionally omitted.
